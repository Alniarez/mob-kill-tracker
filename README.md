MobKillTracker is a World of Warcraft addon that tracks how many times you have killed individual NPCs and displays this information directly on the mob’s tooltip.

Available Commands:

* `/mkt list` - show the kill list window
* `/mkt delete all` - delete all characters data 
* `/mkt delete character` - delete current characters data
* `/mkt delete target` - delete current target NPC data
* `/mkt delete <ID>` - delete a specific NPC by ID"
